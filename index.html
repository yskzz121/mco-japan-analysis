Executive Summary

2026年3月のMr. Sushi Ashmoreは、売上高$105,273（前年比+15.0%、前月比+11.8%）と2月の閑散期から大きく回復し、$100,000超の通常水準に復帰。なお前年Mar-25は台風で3日休業（営業日23日）、本年は26日の営業のため、営業日数を揃えた実質YoYは+1.7%で、1日あたりの売上力は横ばい〜微増が実態。 特筆すべきはデリバリー売上$33,524（前年比+81.9%、全売上の31.8%）の急伸で、Uber Eats単体で+20.4%MoM。1月議事録の「月曜Uber広告ROAS20倍」施策の効果が継続的に表れている。 営業利益は$13,011（営業利益率12.4%、前年比+42.7%）と大幅改善し、目標20%まであと一息の水準。 1月議事録で課題だった労働分配率65%→61.2%へ-3.8pt改善、人件費率も34.8%と業界適正ゾーンの入口に到達。
一方、原価率43.2%（目標40%未満から+3.2pt超過）、食材原価率も29.0%と前月+1.1pt悪化しており、2月議事録で指摘された廃棄・オペレーションロス管理は継続課題。デリバリー比率の上昇もPF手数料負担を押し上げている。 B/Sでは純資産$28,388で債務超過脱出を維持、現預金$90,842・流動比率223%と短期支払能力は十分。次は自己資本比率7.7%（親族借入$242,961依存）の改善が中長期テーマ。

前年同月比 分析サマリー:
売上高は前年比+15.0%（+$13,712）と大幅伸長。ただし前年Mar-25は50年ぶりの台風（サイクロン）で3日間休業（営業日23日）。本年Mar-26は営業日26日のため、営業日数を揃えた実質YoYは +1.7%（$91,560×26/23 = $103,505 vs 実績$105,273）。つまり額面+15.0%の大半は営業日3日増による増収であり、1日あたりの売上力はほぼ横ばい〜微増で読むのが公正。

それでも特筆すべきはデリバリー売上 +81.9%（+$15,096）の急伸で、売上増分のすべてをデリバリーが牽引。Uber広告の継続投下・2025年9月値上げ・新HPのデリバリー導線統合が奏功。 店舗売上は -1.9%（-$1,384）とほぼ横ばい（前年の台風後の駆け込み需要を考えるとむしろ健闘）。

営業利益は +42.7%（+$3,893）と大幅改善。その主因は:
(1) 人件費率の改善（36.7%→34.8%、-1.9pt） — 労働分配率も63.0%→61.2%へ改善し、2024年オープン当初の約100%から着実に低下
(2) 売上規模拡大による固定費レバレッジ — 人件費・家賃等の固定的費用が売上$105k超により希薄化
(3) その他経費の適正化（-82.1%） — 前年にあった保険料・教育費などの一時支出が剥落

要注意点:
(1) 原価率が41.7%→43.2%へ+1.5pt悪化 — デリバリー構成比上昇（20.1%→31.8%）に伴うPF手数料増（売上比10.6%）が主因。ただし食材原価率29.0%は業界水準として健全
(2) 粗利率は-1.5pt低下 — 金額は$6,507増だが、収益構造がより「デリバリー依存型」に変化。継続的な構造モニタリングが必要

損益分岐点の解説:
3月の固定費（経費合計）$46,832に対し、粗利率56.8%で除すと損益分岐点売上は$82,451。
実売上$105,273との差額は$22,822（安全余裕率21.7%）。前月14.6%から大幅に改善し、売上$110kラインを維持できれば25%超の安全圏に。
2024年9月コンサル時のBEP$72,938〜$77,824と比較するとBEP自体は上昇しているが、これは松下氏給与の正規化・3月以降の継続計上（コンサル費・帳簿費等）による構造変化で、むしろ健全化の過程。
→ 営業利益率20%到達のためには、粗利率60%回復（原価率40%未満・1月議事録目標）＋ 売上$110k維持が必要条件
💡 コンサルティング提案
最重要 / 2026年内実行
メニュー全体値上げ（最低賃金上昇・手数料内部化への対応）
3/28コンサルMTGで方針転換。2026年7月からの最低賃金上昇 + カードサーチャージ内部負担化により、人件費率35%目標（将来32%）の維持が厳しくなるため、店内・デリバリー双方を含めたメニュー全体の値上げを実行する方針。
当初案の「Uberのみ5%」は規模不足と判断。松下氏が値上げ率（想定5〜8%）を検討中で、値上げ幅・タイミング・客離れリスク評価を次回論点に。デリバリーPF手数料（売上比10.6%）の吸収原資にもなる。
想定効果: 値上げ幅5%で月$5,000超の粗利改善（営業利益率+4pt超）・賃金上昇耐性確保
最重要 / 継続
食材原価率の深掘り調査（ロス管理）
食材原価率が2月27.9%→3月29.0%へ+1.1pt上昇。前年Mar-25は原価率41.7%（食材部分不明）で基準値がないが、11月以降の食材原価率トレンド（30.3%→29.4%→31.2%→27.9%→29.0%）から、月次ブレ1.5〜2pt程度はある。
2月議事録で指摘された廃棄記録・オペレーションロスの可視化を週次単位で開始。回転寿司レーンの投入量データも記録対象に。
想定効果: 原価率1pt改善で月$1,050の粗利改善
重要 / 短期
DoorDash減少要因の特定
DoorDash売上が前月$3,265→当月$2,037（-37.6%）と唯一大幅減。原因仮説は (1) 広告予算のUber集中、(2) 店舗ランキングの低下、(3) メニュー・写真の陳腐化、(4) DoorDashアプリ上のレビュー問題。
DoorDash管理画面で露出数・クリック数・レーティングを確認し、改善可能であれば4月中に施策実施。
想定効果: DoorDash売上$3,000水準回帰で月$1,000の売上上積み
重要 / 中期
営業利益率20%達成ロードマップ
現状12.4%。20%到達には営業利益$21,000/月（現状比+$8,000）が必要。分解すると:
・売上$110k維持（+$4.7k）→ 粗利+$2.7k
・粗利率57%→60%回帰（原価率改善）→ 粗利+$3.2k
・人件費率34.8%→32%（シフト最適化）→ 営業利益+$3k
3要素すべての達成で射程圏内。
想定効果: 2026年度（11月〜）後半の達成が現実的なタイムライン
最重要 / 方針転換
2店舗目出店は見送り、既存店の収益力・財務体質強化に集中
3/28MTGでの結論: 居抜き物件（既存店1.6倍規模・元寿司屋）の候補はあるが、現状の自己資本比率7.7%で大規模設備投資を実行すると、出店後に自己資本比率4%以下（実質債務超過）に陥るリスクが高い。
「目の前のチャンスより冷静な経営判断」。まずは自己資本比率30%以上・営業利益率15%安定維持を固めてから出店検討に戻る。当面はMr. Sushi Ashmore既存店の強化を最優先に据える。
想定効果: 財務体質の安定・債務超過リスク回避。出店再検討は自己資本比率改善後
重要 / 短期
運転資金借入（$50k〜$5M）の検討・BSシミュレーション
3/28MTGでの論点: 流動比率223%と短期支払能力は十分だが、親族借入$242,961への依存度が高く、外部金融機関借入による運転資金確保 + 保険的手元資金の積み増しが課題。
松下氏が金融機関向けBS共有 → 借入シミュレーション（金額・金利・返済期間）を準備。親族借入の一部返済・整理と並行して進める。
想定効果: 自己資本比率向上のステップ・流動性強化・金融機関取引実績の構築
推奨 / 進行中
Googleレビュー累計約300件達成（★4.7→★4.8へ）
3/28MTG時点で過去2ヶ月で+50件、累計約300件（ほぼ★5中心）を達成し、1月議事録の★4.7目標は射程圏内。次のマイルストーンは★4.8/年内400件へ引き上げ。
レビュー獲得導線（QRコード・会計時声かけ・新HPのGoogleレビュー自動反映）は既に機能中。Uber Eats ★4.8との両輪で、検索流入の上位表示を継続強化。
想定効果: オーガニック流入の自然増・広告費依存度の低下
推奨 / 4月〜実行
新HP + Google広告 + Meta広告 並行運用で集客自動化
3/28MTGで計画確認: 3/30新HP公開 → 同タイミングでGoogle広告開始。既存Instagram広告（AUD10k/月、インプ約11,000、ページ訪問率3.6%、月+300フォロワー、転換140人）と併せてGoogle（検索意図層）＋ Meta（潜在層認知拡大）の並行運用に切替。
Google = 「家族ディナー ゴールドコースト」等の顕在ニーズ、Meta = 認知獲得とリマーケティング、の役割分担でROAS最適化を図る。
想定効果: 広告チャネル分散によるリスク低減・CPA最適化・新規顧客獲得の拡大

総合評価・次回コンサルに向けて

2026年3月は売上$105,273、営業利益$13,011、営業利益率12.4%と、2月の閑散期から完全回復。 特筆すべきはデリバリー売上 前年比+81.9%という構造変化で、1月〜現在にかけて仕込んできたUber広告・HP刷新・Googleレビュー導線の効果が複合的に顕在化した月と評価できる。 労働分配率も65%→61.2%へ改善し、1月議事録で掲げた課題のほぼ全てに前進あり。

一方、原価率43.2%（目標40%超過）と食材原価率29.0%の悪化トレンドは継続監視テーマ。2月議事録で指摘された廃棄・ロス管理の仕組み化が、次の粗利率回復の鍵となる。

ただし、営業日ベースで見ると実質YoY +1.7%（$91,560×26/23 = $103,505 vs $105,273）。額面+15%の大半は営業日3日増による増収で、1日あたりの売上力はほぼ横ばい。ここが実力値の起点。

(3) 2店舗目出店は見送り: 居抜き候補はあるが自己資本比率7.7%では出店後に実質債務超過リスク。Mr.Sushi既存店の強化が最優先。「目の前のチャンスより冷静な経営判断」
